# Games Library Application

The Games Library Application is a simple web application that allows user to browse top-rated and new released games. Allows user to view selected game details

## Getting Started

### Pre-requisites
Before running the application, ensure you have the follow the steps in running the Games API:

- [Games API](https://github.com/merkle-ne-tools/games-api-roijohnkoch)

### Running the application

```bash
# Install dependencies
npm install

# Start the application locally
npm run dev

# Open the application in your browser
# Typically accessible at http://localhost:3000
